# Law Lending - Проект для обучения SCSS

Проект для изучения и практики работы с SCSS/SASS.

## Структура проекта

```
law-lending/
├── scss/              # Исходные SCSS файлы
│   ├── main.scss      # Главный файл (импортирует все остальные)
│   ├── _variables.scss # Переменные
│   ├── _mixins.scss    # Миксины
│   ├── _base.scss      # Базовые стили
│   ├── _layout.scss    # Стили макета
│   └── _components.scss # Компоненты
├── css/               # Скомпилированные CSS файлы (генерируются автоматически)
├── index.html         # HTML файл
└── package.json       # Конфигурация проекта
```

## Установка

1. Установите зависимости:
```bash
npm install
```

## Использование

### Компиляция SCSS в CSS (один раз)
```bash
npm run sass
```

### Автоматическая компиляция при изменении файлов (режим watch)
```bash
npm run sass:watch
```

### Компиляция для продакшена (сжатый CSS)
```bash
npm run sass:prod
```

### Полная сборка проекта
```bash
npm run build
```

## Основные возможности SCSS в проекте

- **Переменные** (`_variables.scss`) - для хранения цветов, размеров, отступов
- **Миксины** (`_mixins.scss`) - переиспользуемые блоки кода
- **Вложенность** - удобная структура стилей
- **Импорты** - модульная организация кода
- **Адаптивность** - миксины для медиа-запросов

## Работа с проектом

1. Откройте `index.html` в браузере
2. Редактируйте файлы в папке `scss/`
3. Запустите `npm run sass:watch` для автоматической компиляции
4. Обновите страницу в браузере для просмотра изменений
